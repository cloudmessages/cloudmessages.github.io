# CloudMessages

![CloudMessages logo](/assets/images/cloudmessages-logo.png)

A normalised message model for modern distributed systems:

- CloudEvents for facts
- CloudCommands for intent
- CloudQueries for information retrieval

CloudMessages aims to improve consistency, interoperability, observability, developer clarity, and scalable message-oriented architecture by making message intent explicit across transports.

For the original project motivation that led to this bootstrap effort, see [Original Prompt](/original-prompt.html).
